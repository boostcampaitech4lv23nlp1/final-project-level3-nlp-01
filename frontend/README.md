## **frontend**

### **installation**

nodejs 웹 서버를 설치합니다.
```bash
$ sudo apt install nodejs
$ sudo apt install npm
$ sudo npm install -g npx
```
<br>

react를 설치합니다.
```bash
npm install -g create-react-app
```

<br>

package.json파일이 위치하는 곳에서 아래의 명령어를 입력하여 실행에 필요한 모듈을 설치합니다.
```bash
$ npm install
```

<br><br><br>

### **usage**

웹 서버를 실행시킵니다. (not build)
```bash
npm start
```
