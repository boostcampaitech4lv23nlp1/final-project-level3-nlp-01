# frontend settings

아래와 같이 명령어 설치 후, terminal을 재시작합니다.
```bash
apt install nodejs
apt install npm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.2/install.sh | bash
```

nvm을 사용하여 node.js 버전 업데이트를 합니다.
```bash
nvm install 17
```

react-native를 설치합니다.
```bash
npm install -g create-react-app
```
